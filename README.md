# Relief - Mental Health Support Website

A mobile-first, compassionate mental health support platform built with React and Vite.

## Features

- **AI Chat Companion**: Empathetic emotional support with voice interaction (Web Speech API).
- **Mood Journal**: Track your daily emotions and thoughts.
- **Guided Exercises**: Visual "Box Breathing" tool for immediate calming.
- **Crisis Resources**: Direct access to 988, 741741, and global helplines.

## Project Structure

- `src/pages`: Individual route components (Home, Chat, Journal, etc.)
- `src/layouts`: Main application shell (Navigation, Header)
- `src/index.css`: Global design system and variables

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Deployment

This project is ready for deployment on platforms like Netlify, Vercel, or GitHub Pages.
Simply connect your repository and set the build command to `npm run build` and publish directory to `dist`.

## Disclaimer

This application is for peer support and self-reflection. It is **not** a diagnostic tool or a replacement for professional medical advice.
